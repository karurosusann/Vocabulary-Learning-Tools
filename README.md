# CSVocab

CSVファイルから暗記カードを作れる、シンプルな単語帳アプリです。

英単語・歴史年号・資格試験対策など、あらゆるジャンルの暗記学習に使えます。  
PC・スマホ・iPadなど、ブラウザがあればどのデバイスでも動作します。

## 特徴

- **CSVで簡単取り込み** — `問題,答え` 形式のCSVファイルを読み込むだけでデッキを作成
- **テキスト貼り付けで追加** — CSVテキストをコピペしてデッキを作成
- **AIで問題作成** — ChatGPT・Gemini・Claudeなどに頼んでCSVを生成できる
- **2つの学習モード** — ノーマルモード（×/○で仕分け）と特訓モード（間違えた問題を繰り返し）
- **タイピングモード** — キーボード入力で回答する学習モード
- **リスニングモード** — 音声読み上げで聞き流し学習
- **4段階レベル管理** — 新規 → 要学習 → あと一歩 → マスター で進捗を可視化
- **画像添付** — 単語カードに画像を添付して視覚的に学習
- **ブックマーク機能** — 気になる問題をピンして後でまとめて復習
- **音声読み上げ** — 問題・回答をタップで読み上げ（ブラウザのTTS機能を使用）
- **カレンダーヒートマップ** — 学習の継続状況をカレンダーで可視化
- **ドーナツチャート・進捗バー** — デッキごとの習熟度を一目で確認
- **一括編集** — 複数の単語を選択してまとめて移動・削除
- **デッキの並べ替え** — ドラッグ＆ドロップでデッキの順序を変更
- **ダークモード対応** — 目に優しいダークテーマに切り替え可能
- **Google Drive同期** — デバイス間で学習データを同期
- **PWA対応** — ホーム画面に追加してアプリのように使える
- **完全オフライン** — データはすべて端末内に保存、サーバー不要

## 使い方

### 1. 問題集を用意する

AIに以下のように頼むと簡単に作れます：

```
「TOEIC頻出英単語100個を『英単語,日本語訳』の形式でCSVにしてください」
「日本史の重要年号50個を『出来事,年号』のCSVにして」
「ITパスポートの用語30個を『用語,意味』のCSVにして」
```

### 2. CSVファイルの形式

1行に「問題,答え」をカンマ区切りで書きます：

```csv
apple,りんご
鎌倉幕府の成立,1185年
TCP/IP,通信プロトコル
```

### 3. 取り込み

ホーム画面の「＋ 新しいデッキを追加」からCSVファイルを選択、またはテキストを貼り付けてデッキを作成できます。

### 4. 学習する

- **ステータスカード** をタップして、特定のレベルの問題だけ学習
- **ノーマルモード** — 「×」「○」で仕分け、正解するとレベルアップ
- **特訓モード** — 間違えた問題を覚えるまで繰り返し出題

## Google Drive同期のセットアップ

デバイス間でデータを同期するには、Google Drive APIの設定が必要です。

### 手順

1. [Google Cloud Console](https://console.cloud.google.com/) を開く
2. プロジェクトを新規作成
3. 「APIとサービス」→「ライブラリ」→「**Google Drive API**」を有効化
4. 「OAuth同意画面」→「外部」で作成 → テストユーザーに自分のGmailを追加
5. 「認証情報」→「認証情報を作成」→「OAuthクライアントID」
6. アプリの種類：「ウェブアプリケーション」
7. **承認済みのJavaScript生成元** に以下を追加：
   - ローカルで使う場合: `http://localhost` など
   - **GitHub Pagesで公開する場合: `https://<ユーザー名>.github.io`**
8. 作成されたクライアントIDをアプリの設定画面に貼り付け

### 使い方

- **進捗を保存する** — 学習が終わったらタップしてGoogle Driveにデータを保存
- **最新データを取得** — 別のデバイスで学習を始める前にタップして最新データを取得

## セットアップ

### GitHub Pagesで公開する場合

1. このリポジトリをforkまたはclone
2. GitHubのリポジトリ設定 → Pages → Source を `main` ブランチに設定
3. `https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセス可能に

### ローカルで使う場合

`index.html` をブラウザで開くだけで動作します。
